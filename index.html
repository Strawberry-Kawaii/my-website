ProGaming - Learning Platform for Programmers

ProGaming is an Interactive Platform that helps users learn programming through gamified challenges, providing real-time process tracking and skill-based rewards.

Problem Statement

After interviewing several classmates, I realized that learning programming often feels static and unengaging. Many learners tend to skip the foundational learning process and jump straight into coding without truly understanding the logic behind it.

As programmers, we love solving problems ‚Äî but we also love having fun while doing it. That insight inspired my team and me to create ProGaming ‚Äî not just another learning platform, but a gamified experience that makes learning to code both exciting and rewarding. We wanted to build something that engages the brain the same way gaming does ‚Äî by turning progress into play.

Goals and Objectives

Primary Goal

- Make programming learning engaging and interactive

Obectives

  • Create a course system with levels and stages
  • Implement tests/quizzes with immediate feedback.
  • Allow users to track progress, save scores, and continue later.
  • Provide a smooth UX across devices.

My Role

Lead Designer

  • Conducted user research to identify challenges in traditional programming education.
  • Designed the platform‚Äôs interface and flow using Figma and FigJam.
  • Created interactive prototypes and gathered user feedback to make immediate design improvements.

Front-end Developer

  • Collaborated with the front-end team to integrate the system with the backend server.
  • Implemented responsive layouts and styled components to reflect live database activities.

Backend Developer

  • Built and deployed the backend server using Node.js and connected it to MongoDB.
  • Implemented MERN-based authentication, ensuring user accounts are secure.
  • Developed a Two-Factor Authentication (2FA) system to strengthen account protection.
  • Integrated third-party APIs such as Brevo and Nodemailer for automated email notifications.
  • Engineered the course progression logic, including streaks, XP points, and gamified tasks to keep users motivated.
